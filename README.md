Student Attendance Management System

Project Description

The Student Attendance Management System is a web-based application used to manage student attendance records efficiently. It allows administrators to add students, mark attendance, and view attendance reports.

Objectives

- Maintain student records digitally
- Mark daily attendance
- Generate attendance reports
- Reduce manual paperwork

Technologies Used

Frontend

- HTML
- CSS
- JavaScript

Backend

- Spring Boot (Java)

Database

- MySQL

Features

- Secure Login
- Student Management
- Attendance Marking
- Dashboard
- Attendance Reports
